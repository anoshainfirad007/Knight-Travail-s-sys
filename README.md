# ♞ Knight Travails

A Knight Travails project that finds the shortest path for a knight piece on a chessboard using JavaScript.

## 🚀 Features
- ♟️ Calculates shortest path for a knight’s movement
- 📍 Works on an 8×8 chessboard grid
- ⚡ Uses efficient algorithms (BFS)
- 🔄 Displays moves from start to end position

## 🛠️ Technologies Used
- Java
